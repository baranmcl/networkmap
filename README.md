# Network Map Project

This project aims to create a PowerBI dashboard visualizing a network map, fed by data from Excel spreadsheets (or CSV files).

## Data Structure

- `data/nodes.csv`: Contains node information (e.g., cities with latitude and longitude).
- `data/edges.csv`: Contains connections between nodes (e.g., roads or links with weights).

## Steps to Build the Dashboard

1. Install PowerBI Desktop from https://powerbi.microsoft.com/desktop/
2. Open PowerBI Desktop.
3. Get data from CSV files in the `data/` folder.
4. Create visualizations: Use map visual for nodes, and line charts or custom visuals for edges.
5. Publish or save the .pbix file.

## Sample Data

The sample data includes major cities in the US with some connections.
